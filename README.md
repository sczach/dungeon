# Dungeon

Lightweight workspace for personal productivity tooling.

- **Job tracking**: CSV-backed tracker with a simple CLI (`scripts/job_tracker.py`) or web UI (`scripts/web_tracker.py`).
- **Email hygiene**: Inbox Zero checklists for Gmail and Outlook (`docs/inbox_zero.md`).
- **Workflows**: Suggestions for tracking applications and follow-ups (`docs/job_acquisition.md`).

## Web UI quick start
1. Install dependencies: `python -m pip install -r requirements.txt`
2. Run the server: `python scripts/web_tracker.py`
3. Open [http://localhost:5000](http://localhost:5000) to add, filter, and update applications without the terminal.
