# Inbox Zero playbook (Gmail + Outlook)

## Daily habit (10–15 minutes)
1. Sort by unread. Convert every email into one of: delete/archive, reply (<2 minutes), or task.
2. For tasks, add to your task manager with a due date and archive the email.
3. Leave only action-needed emails in the inbox by end of day.

## Gmail setup
- Create filters for newsletters/alerts: `Has the words: unsubscribe` → Skip Inbox, Apply label `Newsletters`.
- Create filters for receipts/invoices (keywords: receipt, invoice, order) → Label `Finance`, skip Inbox.
- Starred/Important: turn off auto-importance if it is noisy.
- Keyboard shortcuts: enable in settings; focus on `e` (archive), `r` (reply), `a` (reply all), `s` (star), `/` (search).
- Snooze for day-specific follow-ups so the inbox stays clear.

## Outlook setup
- Use Rules to auto-move newsletters and notifications to a `Newsletters` folder.
- Enable Focused Inbox if it improves signal; otherwise disable and rely on rules + search.
- Quick Steps: create shortcuts for `Archive`, `Flag for Today`, and `Move to Project X`.
- Sweep old newsletters weekly (delete or archive in bulk).

## Weekly cleanup (30–45 minutes)
- Bulk-archive everything older than 30 days in the main inbox (keep Finance/Receipts labeled).
- Unsubscribe from senders that required cleanup this week.
- Review flagged/starred items and convert to tasks with due dates.

## Metrics to track
- Inbox count at end of day.
- Number of follow-ups scheduled from email each week.
- Number of unsubscribe actions taken.
